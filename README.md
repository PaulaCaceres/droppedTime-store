### **dropped-time-store**

## **About The Project**
This is a eCommerce application called "Dropped Time Store".

## **Built With:**
-React
-Material UI
-Firebase

## **Getting Started**
To get a local copy run:
1. `git clone` https://github.com/PaulaCaceres/dropped-time-store.git
2. `yarn install`
